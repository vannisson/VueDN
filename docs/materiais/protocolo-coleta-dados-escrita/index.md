Este documento apresenta as **diretrizes para coleta de dados** em pesquisas sobre escrita colaborativa realizadas pelo LAME.

## Etapas do protocolo

1. Preparação do ambiente de coleta
2. Configuração dos equipamentos de filmagem e gravação
3. Orientações para a díade de alunos
4. Registro do processo de escritura em tempo real
5. Armazenamento e organização dos dados coletados

## Equipamentos recomendados

- Câmera de vídeo posicionada sobre a mesa de escrita
- Gravador de áudio para captura das interações orais
- Software de sincronização áudio-vídeo

O protocolo é utilizado como referência em todas as pesquisas do laboratório.
